# React Portfolio

A data-driven React portfolio inspired by `portfolio.html`.

## Edit Your Content

Update `src/data/portfolio.js` to change your name, projects, skills, experience, and contact links.

## Run Locally

```bash
npm install
npm run dev
```

## Build For GitHub Pages

```bash
npm run build
```

The static output is written to `dist/`. The Vite config uses relative assets, so the build works from GitHub Pages project URLs as well as custom domains.
